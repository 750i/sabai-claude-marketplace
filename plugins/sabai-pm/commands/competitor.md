# /competitor - Competitor Analysis

Create structured competitor analysis using the competitor analysis framework.

## Usage

```
/competitor [company-name]
```

## What It Does

1. Creates structured analysis template
2. Guides research areas
3. Generates comparison matrices
4. Identifies strategic implications

## Research Sources

| Social media | Messaging, engagement |

- Note when they launch features you have
- Use for positioning and messaging
